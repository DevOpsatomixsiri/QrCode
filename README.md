# 🎓 QR Code Attendance System

A simple web-based attendance system using QR codes. Teachers can generate QR codes for students and mark attendance by scanning them.

---

## 📁 Project Structure

```
qr_attendance_system/
├── index.html    → Main page (UI)
├── style.css     → Styling
└── script.js     → Logic (QR generate + scan + attendance)
```

---

## ✨ Features

- ✅ Generate QR Code for each student
- ✅ Scan QR Code to mark attendance
- ✅ View attendance table (Present / Absent)
- ✅ Download QR Code as image
- ✅ Data saved in browser (localStorage)

---

## 🚀 How to Use

1. Open `index.html` in any browser
2. Enter student **Name** and **Roll Number**
3. Click **Generate QR Code**
4. Share the QR with the student
5. Click **Start Scanner** → scan student's QR → attendance marked ✅

---

## 🛠️ Technologies Used

- QR Code JS Library
- html5-qrcode (Scanner)
